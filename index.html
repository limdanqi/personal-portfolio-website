Computer Graphics Assignments

2022 - 2023

The NUS modules CS3241 Computer Graphics and CS4247 Graphics Rendering Techniques have taught me more on the graphics rendering pipeline, modelling, viewports and methods to create a 3D program. Through the assignments, I also got to experiment with concepts such as rasterisation, real-time rendering, illumination, shaders, ray tracing, radiosity and anti-aliasing. At the end of the course, I also managed to create a scene which implements the progressive refinement radiosity algorithm. A wavefront object was then exported from Blender and converted in my program so as to illustrate the scene.

travellless

2022

travellless was a team design project created during the course of CS3240 Interation Design. Its main purpose was to prepare users adequately for travel during Covid-19. In particular, users are provided with a pre-departure checklist and information on their destinations. They are also able to discuss travel tips and organise their documents on this platform. Using Contextual Inquiry, our team conducted user research so that our solution would align with users' needs. I was in charge of designing the checklist and explore page in this project. Being information intensive, the checklist page was structured into a drop down list to declutter and sort information into its respective sub-categories. More information regarding this project can be found here.

StudyTracker

2021

StudyTracker was a team programming project developed during the course of CS2103T Software Engineering. It was a Desktop Application optimized for use via Command Line Interface (CLI). It targets students who wish to track their most productive study spots. During this project, I was in charge of implementing the Amenities and Favourites features. While developing these features, I also had to ensure that data stored was kept up to date with users' inputs. The UI elements and layouts should also be responsive and visually consistent. The Github repo for this project can be found here.

whiteBear

2021

whiteBear was an Android mobile application developed for NUS Orbital 2021. It helps users achieve a healthier lifestyle and mindset through regulating their lifestyles. It features daily reminders, a task list and an entertainment page. I was in charge of the Login, Account Generation and Entertainment page. The app allows users to login via their Google accounts or create a new account with their emails. The Entertainment page showcases different genres of videos for users to relax to. Users can search for a video via a specific keyword or duration. Otherwise, they can select from the different genres and a video would be randomly generated for them.
